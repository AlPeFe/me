Política de Privacidad

La aplicación Racó, desarrollada por Alex Perez, respeta la privacidad de los usuarios. Esta política explica cómo se maneja la información dentro de la aplicación.

1. Recopilación de información

La aplicación Racó no recopila, almacena ni transmite información personal o sensible de los usuarios.

Toda la información generada o introducida por el usuario:

  • Se almacena exclusivamente en el dispositivo del usuario
  • Se guarda en una base de datos local
  • No se envía a servidores externos
  • No se comparte con terceros

2. Uso de la información

Dado que la aplicación no recopila ni transmite datos, la información almacenada localmente se utiliza únicamente para el funcionamiento interno de la aplicación y para ofrecer sus funcionalidades al usuario.

3. Compartición de datos

La aplicación no comparte datos con terceros, ya que no se recopila ni transmite ningún tipo de información.

4. Servicios de terceros

La aplicación no utiliza servicios de terceros que recopilen información del usuario (como servicios de análisis, publicidad o seguimiento).

5. Seguridad

Aunque la información se almacena únicamente de forma local en el dispositivo del usuario, se aplican las medidas técnicas razonables dentro de la aplicación para garantizar su correcto manejo. No obstante, la seguridad del dispositivo depende también del propio usuario.

6. Privacidad de menores

La aplicación no recopila datos personales, por lo que no se realiza ningún tratamiento específico de información de menores.

7. Cambios en esta política

Esta política de privacidad puede actualizarse ocasionalmente. Cualquier cambio se reflejará en esta página con una nueva fecha de actualización.
