# alpefe.me — hand-drawn portfolio

Personal site for [alpefe.me](https://alpefe.me). Hand-drawn aesthetic, soft
pastels, gentle wiggle animations, no frameworks — just HTML, CSS, and a touch
of JavaScript.

## Stack
- Pure HTML5 + CSS3 (Google Fonts: Caveat, Kalam, Patrick Hand)
- Vanilla JS (IntersectionObserver reveal, cursor trail)
- SVG filters (`feTurbulence` + `feDisplacementMap`) for the wobbly hand-drawn
  borders
- DiceBear's `lorelei` style for the placeholder avatar

## Design
- **Paper background** with subtle dot grid + noise texture
- **Pastel palette**: mint, pink, yellow, blue, lavender, peach
- **Wavy borders** generated by SVG displacement filters
- **Continuous wiggle** on tags, social icons, avatar, status badge
- **Reduced-motion friendly**: all animations are disabled under
  `prefers-reduced-motion`
